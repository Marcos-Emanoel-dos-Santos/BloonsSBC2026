# BloonsSBC2026

Repositório voltado para a centralização de soluções para questões de preparação para a Maratona de Programação da SBC 2026.

## Publicação de soluções

A divisão de conteúdo estará na forma [FONTE DA QUESTÃO] -> [ANO_PROBLEMA] -> [PARTICIPANTE.extensao.
Por exemplo, a questão A da Maratona SBC de 2024 resolvida pelo Marcos estará na pasta MaratonaSBC -> 2024_A -> MARCOS.cpp

### Issues

Caso uma questão seja extremamente difícil e ache pertinente pedir ajuda a alguém, poste uma Issue com a questão problemática para que outros participantes tentem resolvê-la.

## Considerações finais

Sincronia entre cada membro da equipe, saber onde cada um excele e onde cada um falha é crucial para a preparação de toda a equipe e obtenção de um bom ranking no dia em que a menor falha custará nossa vitória. Assim podemos nos preparar verdadeiramente como uma equipe que tem o objetivo comum de vencer uma competição de dificuldade elevada, e não como um grupo de programadores individuais.

Tente sempre resolver cada problema como se estivesse sozinho, mas nunca se esqueça de que não está.
